# reports/daily-financial-briefings 目录索引

本文件是 `reports/daily-financial-briefings/` 的目录索引。新增、删除、改名或移动本目录报告时，必须同步更新这里的说明。

## 目录用途

- 保存 `$daily-financial-briefing` 的人工 forward test 和可 review 简报产物。
- 报告必须区分公开来源事实、机构观点、推论、信息缺口和非投资建议边界。
- 本目录产物是 skill 行为验证证据，不是每日行情数据或交易建议。

## 文件索引

- `AGENTS.md`：本目录索引和维护规则。
- `claude.md`：Claude/Codex 入口引用文件，内容固定为 `@agents.md`。
- `forward-test-2026-06-19-public-sources.md`：联网 forward test 正常样例，引用 Fed、BLS、CME、Goldman、Citi 和 UBS 等公开来源生成研究背景简报。
- `forward-test-2026-06-19-no-same-day-bank-views.md`：联网 forward test 缺口样例，验证未找到当日公开投行观点时不编造内容。

## 更新要求

- 新增 forward test 时写明运行日期、检索范围、来源 URL、输出边界和是否触发降级。
- 不把本目录报告写成行情证据、买卖建议或每日复盘结论。
