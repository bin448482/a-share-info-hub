# 来源路由

本文件定义当日财经信息总结 skill 的来源选择和排除规则。第一版不固定抓取源清单，不建设采集管线，只规定 agent 读取公开信息时的优先级。

## 默认主题

### US Macro

覆盖范围：

- 美国 CPI、核心 CPI、PCE、非农、失业率、ADP、初请失业金。
- FOMC、联邦基金利率、利率预期、CME FedWatch 或可验证利率概率转述。
- 美债 2Y、5Y、10Y、30Y 收益率、10Y-2Y 利差。
- 美元流动性代理指标。第一版不强制 DXY；如果没有稳定可引用来源，用美债收益率和利率预期描述替代。

来源优先级：

1. 官方或准官方来源：BLS、BEA、Federal Reserve、U.S. Treasury、CME Group。
2. 可验证数据接口或本地工具：已验证 AKShare 宏观接口、债券收益率接口、FedWatch 数据工具。
3. 主流财经媒体对官方数据或市场预期的转述。

### Investment Bank Views

覆盖范围：

- 高盛、摩根士丹利、摩根大通、花旗、瑞银等主要投行的公开后市判断。
- 对美国宏观、利率、美元、全球风险偏好、中国资产或 A 股相关主题的公开观点。
- 公开财经媒体、研报摘要或投行公开页面中的可引用内容。

来源优先级：

1. 投行公开发布页面、公开文章、公开报告摘要。
2. 主流财经媒体对投行观点的转述。
3. 可访问研报摘要或财经资讯聚合。

第一版不要求每日覆盖每家投行。若当天没有可引用公开投行观点，输出 `未找到可引用的当日公开投行观点`。

## 禁止来源

- 登录后内容。
- 付费墙或订阅墙后的正文。
- 需要验证码、绕过反爬或绕过访问控制的页面。
- 无法给出 URL 或清晰出处的内容。
- 用户未要求的自媒体、微信大 V、社交媒体帖子。
- 来源不明的二次转载。
- 只能看到标题且正文不可验证的内容。

## 使用规则

- 先找原始或权威来源，再用媒体转述补充市场解释。
- 宏观事实和投行观点分开记录，不把观点写成事实。
- 多个来源描述同一事件时合并为一个事实，但保留关键来源。
- 只有标题、摘要不足或无法访问的材料只能进入信息缺口。
- 来源之间冲突时保留冲突和各自来源，不强行消解成单一结论。
