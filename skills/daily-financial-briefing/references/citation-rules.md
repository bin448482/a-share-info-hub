# 引用规则

本文件定义当日财经信息总结 skill 的引用和证据边界。引用用于定位来源，正文必须以摘要和转述为主。

## 必需字段

每个进入核心结论的来源必须包含：

- 来源名称。
- 标题或内容标识。
- URL。
- 发布时间；如果无法取得发布时间，写访问日期。

参考来源格式：

```text
- [来源名称] 标题，发布时间或访问日期，URL
```

核心结论依据格式：

```text
- 依据：[来源名称，发布时间或访问日期，URL]
```

## 引用粒度

- 每条核心结论至少引用一个来源。
- 结合宏观事实和投行观点的结论必须分别引用宏观来源和投行观点来源。
- 同一事实有多个重复来源时优先引用原始或权威来源。
- 媒体转述只证明“媒体如何报道或转述”，不能替代投行原文的存在。

## 不可作为依据的内容

- “市场消息称”“有报道认为”“我看到有人说”等无出处表述。
- 无 URL、无法访问或无法识别来源名称的内容。
- 只有标题可见且无法验证正文的内容。
- 需要登录、付费、验证码或绕过限制才能查看的内容。
- 用户未要求的社交媒体和自媒体观点。

这些内容只能写入 `信息缺口和边界`，不能作为核心结论。

## 版权边界

- 不长段复制来源原文。
- 不摘录完整文章或研报。
- 必要的短引语应保持很短，并且用转述承担主要信息表达。

## 输出前自检

回答前逐条检查：

1. 每条核心结论是否有来源名称、URL、发布时间或访问日期。
2. 宏观事实、市场预期、投行观点和推论是否分开。
3. 投行观点是否被误标为事实。
4. 无法核验的信息是否只在信息缺口中出现。
5. 是否出现交易行动语言。

任一检查失败时，补充来源、降级为信息缺口，或返回 blocked 输出。
