# 输出 Schema

本文件定义当日财经信息总结 skill 的 Markdown 输出结构。输出是研究背景，不是行情证据，也不是投资建议。

## 完整简报

```text
# 当日财经信息简报

- 日期：YYYY-MM-DD
- 范围：US Macro / Investment Bank Views / [用户额外主题]
- 结论性质：研究背景，不构成投资建议

## 1. 核心结论

1. [结论]
   - 依据：[来源名称，发布时间或访问日期，URL]
   - 类型：事实 / 市场预期 / 投行观点 / 推论
   - A 股研究含义：[候选主题、风险观察或待验证问题]

## 2. US Macro

[事实、市场预期、变化方向、来源和边界]

## 3. Investment Bank Views

[公开投行观点、转述来源、缺口或冲突]

## 4. 对 A 股研究的待验证问题

[需要 A 股行情、板块、情绪或事件数据确认的问题]

## 5. 信息缺口和边界

[未发布、不可访问、来源不足、冲突或低可信度部分]

## 6. 参考来源

- [来源名称] 标题，发布时间或访问日期，URL
```

## 核心结论字段

每条核心结论必须包含：

- 结论正文。
- `依据`，至少一个来源。
- `类型`，只能使用 `事实`、`市场预期`、`投行观点`、`推论`。
- `A 股研究含义`，只能写候选主题、风险观察或待验证问题。

禁止在 `A 股研究含义` 中写买入、卖出、加仓、减仓、仓位、目标价、止损、止盈、交易时点或收益承诺。

## 降级输出

如果宏观数据尚未发布：

- 写明发布时间、当前市场预期和待观察指标。
- 不把预测值写成已发布事实。

如果没有可引用投行观点：

- 在 `Investment Bank Views` 写 `未找到可引用的当日公开投行观点`。
- 在信息缺口说明已检查的公开来源类型。
- 不编造机构名称或观点。

如果来源冲突：

- 在核心结论或信息缺口中并列写出冲突点。
- 分别列出来源，不自行裁决真伪。

## Blocked 输出

无法联网、无法访问外部信息、无法获得 URL，或用户没有提供可引用材料时，不输出完整简报。使用：

```text
blocked: 缺少可引用公开来源，无法生成当日财经信息简报。
日期：YYYY-MM-DD
范围：US Macro / Investment Bank Views
缺口：[具体缺少的信息或来源类型]
下一步：[需要联网、指定来源、提供材料或改为离线 fixture]
```
